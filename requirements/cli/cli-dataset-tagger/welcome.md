# cli-dataset-tagger

Please generate requirements document using [prompts](../../supporting/prompts/cli.md)

For every page of Arabic Dictionary dataset [lisan-ul-arab](https://github.com/zawjen/ds-lisan-ul-arab-qum-1405h-hawza), write a Python CLI to

- Tag heading and define boundary
- Tag Word and define boundary
- Meaning and define boundary
- Page number
- Volume number

- Install [gImageReader](https://github.com/manisandro/gImageReader/releases/download/v3.4.2/gImageReader_3.4.2_qt5_i686.exe) to extract page text
